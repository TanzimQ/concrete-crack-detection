# Concrete Crack Detection (Computer Vision + Deep Learning)

## Overview
This project compares traditional computer vision (SIFT + SVM) with deep learning (ResNet50) for automated concrete crack detection.

## Results
- SIFT + SVM Accuracy: 99.88%
- ResNet50 Accuracy: 99.78%

## Key Insights
- Traditional methods can outperform deep learning in certain cases
- SIFT works well due to edge-based crack features
- Faster and CPU-friendly compared to deep learning

## Technologies Used
- Python
- OpenCV
- Scikit-learn
- PyTorch
